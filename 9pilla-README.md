# 9Pilla — Controle de Investimentos

App pessoal de controle de carteira de investimentos.

## Como publicar no GitHub Pages (5 passos)

1. Acesse [github.com](https://github.com) e faça login (ou crie uma conta grátis)
2. Clique em **"New repository"** → nome: `9pilla-app` → marque **Public** → clique **Create repository**
3. Na página do repositório, clique em **"uploading an existing file"** → arraste o arquivo `index.html` → clique **Commit changes**
4. Vá em **Settings** → **Pages** → em *Branch* selecione `main` → clique **Save**
5. Aguarde ~1 minuto → seu app estará em: `https://SEU-USUARIO.github.io/9pilla-app`

## Como atualizar o app toda semana

1. Abra o Claude e envie os arquivos Warren (PDF + CSV)
2. O Claude gera um novo `index.html` atualizado
3. No GitHub, abra o repositório → clique no arquivo `index.html` → ícone de lápis (editar) → cole o novo conteúdo → **Commit changes**

Pronto! O link continua o mesmo, atualizado automaticamente.
